여러 옵시디언 보관소에 대한 바로가기 아이콘 만들기 - URI

게시 2024/01/08
By Uncle's Note 3 분읽는 시간

경우에 따라 카테고리에 따라 여러 개의 볼트를 생성하여 게시물을 정리하는 사용자가 있을 수 있습니다. 옵시디언에서 보관소를 열 때마다 여는 것은 귀찮을 수 있습니다. 그래서 윈도우 환경에서 볼트별로 옵시디언을 열 수 있는 바로가기 아이콘을 생성하는 방법을 소개하고자 합니다.

원리를 설명하자면, 바로가기 아이콘으로 obsidian.exe를 실행할 때 매개변수를 인수 값으로 추가하여 옵시디언에게 먼저 열 볼트를 알려줍니다. 옵시디언은 이것을 URI라고 부릅니다.

미리 알아야 할 사항

옵시디언 보관소 바로가기 아이콘을 만들기 전에 obsidian.exe 경로와 보관소 경로를 알아야 합니다. 결론적으로 아래 형태의 명령줄을 바로가기 아이콘으로 설정해보겠습니다.

1

<Path Of Obsidian.exe> obsidian://open?path=<Path Of Vault>

아래 예시를 보면 Target 명령줄을 쉽게 파악할 수 있습니다.

# Example 
<Path of Obsidian.exe>
    C:\Users\myaccount\AppData\Local\Obsidian\Obsidian.exe
    
<Path Of Vault>
    C:\Data\VaultExport
    obsidian://open?path=C:\Data\VaultExport

# Target 
C:\Users\myaccount\AppData\Local\Obsidian\Obsidian.exe obsidian://open?path=C:\Data\VaultExport

옵시디언 바로가기 아이콘 생성

옵시디언 보관소 바로가기 아이콘을 생성하려는 위치를 마우스 오른쪽 버튼으로 클릭하고 바로가기 버튼을 누릅니다.

옵시디언 바로가기 아이콘 생성 과정 - 바로가기 생성

그런 다음 obsidian.exe의 전체 경로를 입력합니다.

옵시디언 바로가기 아이콘 생성 과정 - obsidian.exe 경로

그런 다음 바로가기 아이콘의 이름을 입력하세요.

옵시디언 바로가기 아이콘 생성 과정 - 아이콘 이름

그러면 입력한 이름의 옵시디언 바로가기 아이콘이 생성됩니다. 그리고 아이콘을 마우스 오른쪽 버튼으로 클릭하고 속성 메뉴를 선택합니다.

생성된 옵시디언 바로가기 아이콘의 속성 메뉴

obsidian.exe 경로 뒤에 공백 하나를 추가하고 위에서 생성한 옵시디언 보관소 경로를 추가합니다.
