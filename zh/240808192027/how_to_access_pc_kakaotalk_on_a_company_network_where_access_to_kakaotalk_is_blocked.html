如何在 KakaoTalk 访问被阻止的公司网络中连接到 PC KakaoTalk

发表于 2024/08/08
作者 Uncle's Note 7 分钟阅读

在公司或特殊的本地网络环境中，可能存在无法访问PC版KakaoTalk应用程序的情况。原因是公司网络层面对 KakaoTalk 服务器的访问被阻止。理论上，您可以使用代理或 VPN 访问 KakaoTalk 服务器。然而，就我而言，即使是已知的代理服务器和 VPN 服务也被阻止访问公司网络。

因此，我想到可以通过笔记本电脑上的个人智能手机的热点 Wi-Fi 连接到 KakaoTalk。恩，那就对了。 PC KakaoTalk 应用程序可以从该公司的 LabTap 访问，该应用程序使用个人热点网络。但无法访问公司内网。

笔记本电脑常见的网络连接方式包括以太网(有线局域网)和Wi-Fi。因此，我想到了一个好主意：使用公司内部网和允许通过以太网访问的互联网站点，并通过 Wi-Fi 连接到个人智能手机的热点来使用 KakaoTalk。然而，当有线局域网和Wi-Fi同时连接时，要使用的网络的优先级仅适用于最先连接的网络。也就是说，如果您将公司的有线局域网连接到以太网端口，然后再连接Wi-Fi 热点，将无法访问 KakaoTalk 应用程序，您将使用最初连接的公司以太网网络。因此，由于公司网络规则，无法访问 KakaoTalk 服务器。相反，如果您先连接Wi-Fi，您将能够访问KakaoTalk应用程序，但无法使用公司内网。此外，尽管可以通过单独的身份验证通过外部网络访问公司内网，但由于是通过个人热点连接，因此智能手机数据使用量会增加。


所以，我想分享一下如何使上述想法成为可能。这个想法是只允许 KakaoTalk 应用程序使用您的智能手机热点所连接的 Wi-Fi 网络。本质上，这似乎是一个神奇的提议，但它会耗尽您的个人互联网数据，并且可以在以下情况下使用。您的智能手机远离笔记本电脑的缺点是 Wi-Fi 连接丢失，您必须返回并重新登录 KakaoTalk 应用程序。

ForceBindIP 首选项

事实上，绑定应用程序以使用特定网络接口的库在 r1ch.net 上共享。但是，由于它是一个命令行实用程序库，ixjb94 提供了一个 GUI 应用程序以使其更易于使用。

1.下载并安装ForceBindIP-1.32库

为了保证 GUI 应用程序和库之间的兼容性，请从 ixjb94 的 GitHub 存储库下载 ForceBindIP-1.32 库。

  • https://github.com/ixjb94/ForceBindIP-Gui/blob/main/ForceBindIP/ForceBindIP-1.32.zip

ForceBindIP-1.32.zip 中包含的文件

然后，将 zip 文件解压缩到适当的位置，并将其中的四个文件复制到下面的文件夹中。

1
2

C:\Windows\system32
C:\Windows\sysWOW64

2. 使用 ForceBindIP-1.32 库设置 GUI 应用程序

如果您在上述步骤中将库复制到系统中，请从下面下载 ixjb94 开发的 GUI 应用程序。

  • https://github.com/ixjb94/ForceBindIP-Gui/releases/download/binary/ForceBindIP.-.Gui.exe

ForceBindIP.-.Gui.exe 实际上是一个 GUI 应用程序，用于将 KakaoTalk 应用程序配置为使用 Wi-Fi 网络。没有什么需要特别设定的。

使用ForceBindIP

1. 连接以太网（有线局域网、公司网络）和Wi-Fi（个人热点）

连接到以太网的公司网络成为主要网络非常重要。首先，拔掉以太网线并断开 WiFi 连接。然后将公司的有线局域网连接到以太网端口。确认互联网正常工作后，打开智能手机热点并连接到笔记本电脑上的 Wi-Fi。

2. 关闭KakaoTalk应用程序

我们将通过设置与ForceBindIP.-.Gui.exe一起使用的网络接口来运行 KakaoTalk 应用程序。如果 KakaoTalk 已在运行，则 KakaoTalk 应用程序将因ForceBindIP.-.Gui.exe而无法运行。因此，如果 KakaoTalk 应用程序正在运行，请关闭该应用程序。右键单击窗口右下角的 KakaoTalk 应用程序图标将其关闭。

窗口右下角 > 右键单击​​ KakaoTalk 图标 > 退出

3.运行KakaoTalk应用程序

运行在上述步骤中下载的ForceBindIP.-.Gui.exe。

A 是应用程序 B 选择要使用的网络接口的地方。选择 Wi-Fi 网络适配器。适配器列表因您使用的 PC 环境而异。

ForceBindIP.-.Gui.exe

然后在B中输入KakaoTalk安装路径的exe文件。

1

C:\Program Files (x86)\Kakao\KakaoTalk\KakaoTalk.exe

在 C 中选择 x86，然后按 D 中的运行应用程序按钮。

KakaoTalk 应用程序可以正常登录吗？如果进展顺利的话，我会给你一个提示。连接到智能手机热点的 Wi-Fi 时，选择自动连接复选框。如果您携带智能手机离开房间，Wi-Fi 连接将被断开，KakaoTalk 将被注销。当您返回座位时，Wi-Fi 将自动连接。此时，如果您已退出KakaoTalk，只需重新登录即可。
