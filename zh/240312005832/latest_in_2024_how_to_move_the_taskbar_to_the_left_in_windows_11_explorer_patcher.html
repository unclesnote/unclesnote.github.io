2024 年最新 - 如何在 Windows 11 中将任务栏向左移动 - Explorer Patcher

发表于 2024/03/12
作者 Uncle's Note 6 分钟阅读

如果您一直在使用 Windows 10 或者熟悉 Ubuntu 任务栏，您可能会失望地发现 Windows 11 中无法移动任务栏的位置。

就我而言，我进行软件编码，我想将任务栏移至左侧以确保显示屏底部的空间并创建一个舒适的编码环境。那么，我想介绍一下如何在2024年最新的Windows 11中将任务栏移至左侧。

注册表值修改方法-不起作用

更改任务栏位置 - 修改注册表不再有效

恩，那就对了。通过修改注册表来更改任务栏的位置不再有效。随着 Windows 11 的更新，这似乎变得不可能了。作为替代方案，Windows 11 中默认没有提供已知的方法。但是，有一些软件可以让您配置任务栏。

《Start 11》的售价为每份 10 美元，我还没有尝试过。与每个人一样，付费软件的安装可能很麻烦。

  • Start 11：付费(每份约 10 美元)，提供 30 天评估期
  • Explorer Patcher：免费开源(分布在 GitHub 上)

相反，Explorer Patcher有免费政策。软件发布在 GitHub 上，如果你查看历史记录，就会发现频繁的代码修改。这可能意味着该软件还不稳定，也可能是它正在快速发展。所以，我决定押注于后者并尝试一下。

Windows 10 和 Windows 11 任务栏比较

您可能突然想知道是否要比较 Windows 10 和 Windows 11 的任务栏。原因是我们决定使用的Explorer Patcher软件试图遵循Windows 10如何操作任务栏的概念。

关于任务栏内容对齐功能，Windows 10默认为左对齐，且对齐方式无法更改。不过，Windows 11的任务栏内容在发布之初仅支持居中对齐，但现在可以在设置中更改为左对齐。

物品Windows 10视窗11
任务栏内容排序左对齐提供左对齐和居中对齐选项
任务栏位置可以左、右、上、下移动固定在下面

关于任务栏位置，Windows 10 允许您将其向上、向下、向左或向上移动。不过Windows 11中任务栏的位置是固定在底部的，在官方设置中无法更改。

将任务栏内容左对齐
Explorer Patcher软件遵循Windows 10的操作理念，只允许任务栏内容左对齐。如果您决定使用软件来移动任务栏，则此提示毫无意义。通常，更改任务栏内容对齐方式的设置位于设置 > 个性化 > 任务栏行为 > 任务栏对齐方式中。

如上所述，这款软件有一个遵循Windows 10任务栏操作方法的概念。如果将任务栏样式设置为 Windows 10，则没有像 Windows 11 那样使任务栏内容居中的选项。
